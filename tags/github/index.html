github clone 速度慢解决方案

2021-12-27

有时候从github上clone项目速度很慢，可以将链接中域名github.com替换成github.com.cnpmjs.org,比如原先的clone url为

https://github.com/wufan123/xxxx.git

替换为

https://github.com.cnpmjs.org/wufan123/xxxx.git
